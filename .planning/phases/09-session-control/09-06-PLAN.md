**Create `packages/web/src/hooks/useAuth.ts`:**

State: AuthState (from interfaces above).
Initialize from localStorage on mount (read all 4 keys → set isAuthenticated = !!accessToken).

`login(tokens, user, project)`:
- Save to localStorage: gantt_access_token, gantt_refresh_token, gantt_user, gantt_project
- Update state: isAuthenticated=true, user, project, accessToken
- Fetch projects list: GET /api/projects (Authorization: Bearer token) → set state.projects

`logout()`:
- Remove all 4 localStorage keys
- Reset state to empty

`refreshAccessToken()`:
- POST /api/auth/refresh with body { refreshToken: localStorage.getItem('gantt_refresh_token') }
- On 200: update gantt_access_token in localStorage + state.accessToken, return new token
- On error: call logout(), return null

`switchProject(projectId)`:
- POST /api/auth/switch-project with Bearer header, body { projectId }
- On 200: receive { accessToken, refreshToken } — update localStorage + state.accessToken + state.project
- Reload tasks: trigger by updating state.project (App.tsx useTasks depends on it)

`createProject(name)`:
- POST /api/projects with Bearer header, body { name }
- On 200: receive new project → add to state.projects, return project
- Returns null on error
