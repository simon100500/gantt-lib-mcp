JWT payload shape:
```typescript
// packages/server/src/auth.ts
export interface JwtPayload {
  sub: string;      // user_id
  email: string;
  projectId: string;
  sessionId: string;
  type: 'access' | 'refresh';
}

export function signAccessToken(payload: Omit<JwtPayload, 'type'>): string
export function signRefreshToken(payload: Omit<JwtPayload, 'type'>): string
export function verifyToken(token: string): JwtPayload  // throws on invalid
export function generateOtp(): string  // returns 6-digit string like '047821'
```

AuthStore operations:
```typescript
// packages/mcp/src/auth-store.ts
export class AuthStore {
  // OTP lifecycle
  async createOtp(email: string, code: string): Promise<OtpEntry>
  async consumeOtp(email: string, code: string): Promise<boolean>  // marks used=1, returns false if not found/expired/used

  // User lifecycle
  async findOrCreateUser(email: string): Promise<User>

  // Project lifecycle
  async createDefaultProject(userId: string): Promise<Project>
  async listProjects(userId: string): Promise<Project[]>
  async createProject(userId: string, name: string): Promise<Project>

  // Session lifecycle
  async createSession(userId: string, projectId: string, accessToken: string, refreshToken: string): Promise<Session>
  async findSessionByAccessToken(token: string): Promise<Session | undefined>
  async findSessionByRefreshToken(token: string): Promise<Session | undefined>
  async updateSessionTokens(sessionId: string, accessToken: string, refreshToken: string): Promise<void>
  async deleteSession(sessionId: string): Promise<void>
}

export const authStore: AuthStore;
```

REST endpoint contracts (all return JSON):
```
POST /api/auth/request-otp
  body: { email: string }
  response 200: { sent: true }
  response 400: { error: 'email required' }
  response 500: { error: 'Failed to send email' }

POST /api/auth/verify-otp
  body: { email: string, code: string }
  response 200: { accessToken: string, refreshToken: string, user: { id, email }, project: { id, name } }
  response 400: { error: 'Invalid or expired code' }

POST /api/auth/refresh
  body: { refreshToken: string }
  response 200: { accessToken: string, refreshToken: string }
  response 401: { error: 'Invalid refresh token' }

POST /api/auth/logout
  headers: Authorization: Bearer <accessToken>
  response 200: { ok: true }
  response 401: { error: 'Unauthorized' }
```

Email env vars (read from process.env):
```
EMAIL_HOST      — SMTP host (e.g., smtp.gmail.com)
EMAIL_PORT      — SMTP port (default: 587)
EMAIL_USER      — SMTP username
EMAIL_PASS      — SMTP password
EMAIL_FROM      — sender address (e.g., 'Gantt App <noreply@example.com>')
```

JWT env vars:
```
JWT_SECRET      — signing secret (required, throw if missing)
```

Create `packages/server/src/routes/auth-routes.ts` as a Fastify plugin:
- `export async function registerAuthRoutes(fastify: FastifyInstance)`
- POST /api/auth/request-otp:
  1. Validate email present
  2. generateOtp()
  3. authStore.createOtp(email, code)
  4. sendOtpEmail(email, code) — catch errors, return 500 if fails
  5. Return { sent: true }
- POST /api/auth/verify-otp:
  1. Validate email + code
  2. authStore.consumeOtp(email, code) — if false return 400 { error: 'Invalid or expired code' }
  3. authStore.findOrCreateUser(email)
  4. authStore.listProjects(user.id) — if empty, createDefaultProject
  5. Take first project (or the new default)
  6. signAccessToken + signRefreshToken with { sub: user.id, email, projectId, sessionId: randomUUID() }
  7. authStore.createSession(user.id, project.id, accessToken, refreshToken)
  8. Return { accessToken, refreshToken, user: { id, email }, project: { id, name } }
- POST /api/auth/refresh:
  1. Validate refreshToken in body
  2. authStore.findSessionByRefreshToken(refreshToken) — if not found return 401
  3. verifyToken(refreshToken) — if throws return 401
  4. Sign new access + refresh tokens
  5. authStore.updateSessionTokens(session.id, newAccess, newRefresh)
  6. Return { accessToken, refreshToken }
- POST /api/auth/logout:
  1. Extract Bearer token from Authorization header
  2. authStore.findSessionByAccessToken(token) — if not found return 401
  3. authStore.deleteSession(session.id)
  4. Return { ok: true }

Register routes in `packages/server/src/index.ts`:
- Import registerAuthRoutes
- Call `await registerAuthRoutes(fastify)` before the existing route registrations
