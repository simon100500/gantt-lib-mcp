  3. Add condition after fetching task:
     ```typescript
     let children: Task[] = [];
     if (includeChildren !== false) {
       const childTasks = await this.prisma.task.findMany({
         where: { parentId: id },
         include: { dependencies: true },
         orderBy: { sortOrder: 'asc' },
       });

       children = childTasks.map(child =>
         this.taskToDomain(child, child.dependencies.map(/* existing mapping */), true)
       );

       // Recursive loading for 'deep' mode
       if (includeChildren === 'deep') {
         for (const child of children) {
           const nestedChildren = await this.get(child.id, 'deep');
           child.children = nestedChildren.children || [];
         }
       }
     }
     ```

  **Circular reference prevention:** The recursive call already exists at the same level, so no cycle detection needed (children won't include their own children again unless 'deep').

  **Type update:** Add `children?: Task[]` to Task interface in types.ts (Task already allows optional properties via index signature, but add explicit field for clarity).
