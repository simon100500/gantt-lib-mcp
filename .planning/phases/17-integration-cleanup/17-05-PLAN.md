# Phase 17-05: gantt-lib 0.8.0 Migration

**Status:** pending
**Wave:** 5
**Autonomous:** true

## Objective

Migrate to gantt-lib 0.8.0 with new `onTasksChange` API that sends only changed tasks instead of full array. This fixes the PUT /api/tasks bug that caused DELETE ALL + INSERT ALL on every drag.

## Background

**Old API (0.7.x):**
```typescript
onChange?: (tasks: Task[] | ((prev: Task[]) => Task[])) => void;
```
- Received full tasks array on every change
- Frontend sent PUT /api/tasks with all tasks â†’ DELETE ALL + INSERT ALL â†’ 500 errors

**New API (0.8.0):**
```typescript
onTasksChange?: (tasks: Task[]) => void;  // Only changed tasks
onPromoteTask?: (taskId: string) => void;
onDemoteTask?: (taskId: string, newParentId: string) => void;
```
- Receives ONLY changed tasks (array of 1 for single edit, multiple for cascade)
- Frontend can send individual PATCH requests
- No more bulk PUT on every drag

### Task 2: Update App.tsx handlers
**File:** `packages/web/src/App.tsx`

**Replace `handleTaskChange` with `handleTasksChange`:**
```typescript
const handleTasksChange = useCallback(async (changedTasks: Task[]) => {
  if (!auth.isAuthenticated || hasShareToken) {
    // Guest mode: merge changed tasks into state
    setTasks(prev => {
      const changedMap = new Map(changedTasks.map(t => [t.id, t]));
      return prev.map(t => changedMap.get(t.id) ?? t);
    });
    return;
  }

  // Authenticated mode: individual PATCH requests
  const originalTasks = changedTasks.map(t => ({ task: t, original: tasks.find(orig => orig.id === t.id) }));

  try {
    // Optimistic update
    setTasks(prev => {
      const changedMap = new Map(changedTasks.map(t => [t.id, t]));
      return prev.map(t => changedMap.get(t.id) ?? t);
    });

    // Send PATCH for each changed task
    await Promise.all(changedTasks.map(t => mutateTask(t)));
  } catch (error) {
    console.error('[App] Failed to update tasks:', error);
    // Revert on error
    setTasks(prev => {
      const revertMap = new Map(
        originalTasks.filter(({ original }) => original)
          .map(({ task, original }) => [task.id, original])
      );
      return prev.map(t => revertMap.get(t.id) ?? t);
    });
  }
}, [auth.isAuthenticated, hasShareToken, mutateTask, tasks, setTasks]);
```
