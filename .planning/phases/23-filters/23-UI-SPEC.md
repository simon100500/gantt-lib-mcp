## Layout & Interaction

### Filter Button Placement

- Location: Toolbar, right side, between viewMode switcher and Ellipsis menu
- Icon: `Funnel` from lucide-react (h-4 w-4)
- Size: h-7 height (matching other Toolbar buttons)
- Variant:
  - No active filters: `variant="ghost"` with `text-slate-600 hover:text-slate-900`
  - Has active filters: `variant="secondary"` or `bg-secondary text-secondary-foreground`

### Filter Popup Structure

```
DropdownMenuContent align="end" className="w-56"
├── DropdownMenuLabel (optional header)
├── DropdownMenuSeparator
├── Filter: Without deps (checkbox)
├── Filter: Expired (checkbox)
├── DropdownMenuSeparator
├── Search: Input (text)
├── DropdownMenuSeparator
├── Date Range: From (date input)
├── Date Range: To (date input)
```

### Visual Hierarchy

1. Section separators: `DropdownMenuSeparator` between filter groups
2. Checkbox items: Use `DropdownMenuItem` with inline `<input type="checkbox">` pattern (see existing Toolbar)
3. Text input: Full width inside dropdown content
4. Date inputs: Side-by-side with labels
5. Reset button: Full width, `variant="destructive"` or ghost with muted text

## State Management

### useUIStore Extension

Add to `UIState` interface:

```typescript
// Filter state
filterWithoutDeps: boolean;
filterExpired: boolean;
filterSearchText: string;
filterDateFrom: string;  // ISO date string YYYY-MM-DD
filterDateTo: string;    // ISO date string YYYY-MM-DD

// Actions
setFilterWithoutDeps: (value: boolean) => void;
setFilterExpired: (value: boolean) => void;
setFilterSearchText: (value: string) => void;
setFilterDateFrom: (value: string) => void;
setFilterDateTo: (value: string) => void;
resetFilters: () => void;
```

### Computed Filter Predicate

```typescript
// Derived value in component or store
const taskFilter = useMemo(() => {
  const predicates = [];
  if (filterWithoutDeps) predicates.push(withoutDeps());
  if (filterExpired) predicates.push(expired());
  if (filterSearchText) predicates.push(nameContains(filterSearchText));
  if (filterDateFrom && filterDateTo) {
    predicates.push(inDateRange(new Date(filterDateFrom), new Date(filterDateTo)));
  }
  return predicates.length > 0 ? and(...predicates) : undefined;
}, [filterWithoutDeps, filterExpired, filterSearchText, filterDateFrom, filterDateTo]);
```

## Persistence

### localStorage

- Key: `gantt-filters`
- Schema: JSON object with all filter fields
- Load on mount: Read from localStorage, populate useUIStore
- Save on change: Write to localStorage on any filter state change

```typescript
const FILTER_STORAGE_KEY = 'gantt-filters';

// Load
const savedFilters = localStorage.getItem(FILTER_STORAGE_KEY);
if (savedFilters) {
  const { filterWithoutDeps, filterExpired, filterSearchText, filterDateFrom, filterDateTo } = JSON.parse(savedFilters);
  // Set initial state
}

// Save
useEffect(() => {
  localStorage.setItem(FILTER_STORAGE_KEY, JSON.stringify({
    filterWithoutDeps,
    filterExpired,
    filterSearchText,
    filterDateFrom,
    filterDateTo,
  }));
}, [filterWithoutDeps, filterExpired, filterSearchText, filterDateFrom, filterDateTo]);
```
