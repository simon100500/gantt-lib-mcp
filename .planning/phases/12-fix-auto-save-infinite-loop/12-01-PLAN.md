# Plan 01: Fix auto-save infinite loop and optimize auth

**Phase:** 12-fix-auto-save-infinite-loop
**Plan:** 01
**Type:** execute
**Wave:** 1

## Objective

Fix the infinite loop bug where PUT /api/tasks requests fire every ~1 second even when user is idle. Also optimize auth middleware to reduce database load.

**Purpose:** The bug started after quick-020 enabled autoSchedule=true by default. Gantt-lib's onCascade callback triggers onChange with new array references even when data hasn't changed, causing useAutoSave to fire repeatedly.

**Output:** Stable auto-save that only fires on actual data changes, faster auth with session caching, cleaner logs.

## Root Cause (from CONTEXT.md)

1. **useAutoSave triggers on any array reference change** - no deep comparison
2. **Gantt-lib calls onChange after cascade** - new array, same data
3. **handleCascade merges tasks** - creates new array reference
4. **Loop:** onChange → setTasks → useAutoSave → PUT → response → onChange

**files:** `packages/web/src/hooks/useAutoSave.ts`

**action:**
Add deep equality check before triggering save. Only save if task data actually changed, not just array reference.

### Task 2: Add session cache to authStore

**files:** `packages/mcp/src/auth-store.ts`

**action:**
Add in-memory cache to avoid database query on every authenticated request.

Add to AuthStore class:
```typescript
private sessionCache = new Map<string, { session: Session; expiresAt: number }>();
private readonly CACHE_TTL_MS = 5 * 60 * 1000; // 5 minutes

async findSessionByAccessToken(accessToken: string): Promise<Session | undefined> {
  // Check cache first
  const cached = this.sessionCache.get(accessToken);
  if (cached && cached.expiresAt > Date.now()) {
    return cached.session;
  }
  // Cache miss or expired - query DB
  const session = await this.dbQuery(accessToken);
  if (session) {
    this.sessionCache.set(accessToken, {
      session,
      expiresAt: Date.now() + this.CACHE_TTL_MS
    });
  }
  return session;
}

// Clear cache entry on token refresh or logout
clearSessionCache(accessToken: string): void {
  this.sessionCache.delete(accessToken);
}
```
