# Phase quick-260321-m63 Plan 01: Add Create Task Button Summary

Добавлена кнопка "+ задача" справа от поля поиска для быстрого создания новых задач.

## One-liner
Кнопка создания задач в строке поиска с optimistic update и синхронизацией с сервером для авторизованных пользователей.

## Implementation Details

### Changes Made

**1. TaskSearch.tsx enhancements:**
- Импортированы `useAuthStore`, `useTaskMutation` и тип `Task`
- Добавлен доступ к `accessToken`, `setTasks` и функции `createTask`
- Реализована функция `handleCreateTask` с optimistic update
- Добавлена кнопка "+ задача" справа от поля поиска

**2. Функционал создания задач:**
- Имя задачи берётся из `searchQuery.trim()` или используется "Новая задача"
- Дата задачи устанавливается на сегодня (startDate = endDate = today)
- Optimistic update: задача сразу добавляется в UI
- Для авторизованных пользователей: задача синхронизируется с сервером
- При ошибке сервера: задача откатывается из UI
- После создания поле поиска очищается

**3. UI изменения:**
- Контейнер поиска теперь использует `gap-2` для отступа между полем и кнопкой
- Кнопка имеет высоту `h-9` (как поле ввода)
- Стиль: `text-indigo-600 hover:text-indigo-700 hover:bg-indigo-50`
- Кнопка имеет `shrink-0` для предотвращения сжатия
