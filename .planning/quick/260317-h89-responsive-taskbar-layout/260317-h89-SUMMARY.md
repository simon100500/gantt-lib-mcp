## What Was Built

### Gantt Toolbar Responsiveness
- Toggle switches ("Закрепить связи", "Просроченные") wrapped in `hidden xl:flex` — invisible below 1280px
- Three-dots overflow `DropdownMenu` added with `xl:hidden` — appears only on medium screens
- Overflow menu items show toggle state via colored dot (emerald = on, slate = off)
- `onSelect` uses `e.preventDefault()` to keep dropdown open after toggle
- Toolbar height changed from `h-11` to `h-auto min-h-[2.75rem]` supporting graceful flex-wrap

### Top Bar Condensation
- "Войдите, чтобы сохранить график" text: `hidden lg:inline` (hidden below 1024px)
- Share button text ("Поделиться"/"Скопировано"): `hidden lg:inline` — icon remains visible
- Email dropdown: `max-w-[180px] lg:max-w-[280px]` for tighter truncation on medium screens
- "+ Новый проект" button: `hidden md:inline-flex` (hidden below 768px)

### Wide Screen Behavior (xl+)
All controls remain inline — overflow menu is hidden, layout identical to before.
