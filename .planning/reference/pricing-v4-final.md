- Team месяц: без изменений (4 990 ₽)

- Enterprise месяц: без изменений (12 900 ₽)

### Лимиты проектов
- Free: 2 → 1 активный (без архива, удалить чтобы создать новый)
- Start: 5 → 3 активных (с архивом)
- Team: 20 → 7 активных (с архивом)
- Enterprise: без изменений (безлимит)

### AI-запросы (ключевое изменение)
- Старая модель: отдельно "генерации" и "уточнения" (месячный лимит)
- Новая модель: единая метрика "AI-запросы"
- Free: 20 запросов ВСЕГО (не сбрасывается, одноразовый бюджет)
- Start: 25/день (сброс в полночь МСК)
- Team: 50/день
- Enterprise: 100/день

### Участники
- Team: 5 (без изменений)
- Enterprise: 20 (было 20+, убрали "+")

### Новые фича-гейты
- Архив проектов: нет на Free, да на платных
- Пул ресурсов: нет на Free, да на платных
- Экспорт: нет на Free, PDF на Start, PDF+Excel на Team, PDF+Excel+API на Enterprise

## Логика пути пользователя

1. **Free → Start**: упёрся в 1 проект или кончились 20 AI-запросов
2. **Start → Team**: нужно больше 3 проектов или 25 AI-запросов/день мало для активной работы, нужна команда
3. **Team → Enterprise**: больше 7 проектов, больше 5 участников, нужен API
